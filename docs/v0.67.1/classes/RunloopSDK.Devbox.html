    Class Devbox

    Object-oriented interface for working with Devboxes.

    Overview

    The Devbox class provides a high-level, object-oriented API for managing devboxes. It wraps the low-level API client and provides convenient methods for common operations.

    Creating Devboxes

    Basic Creation

    import { RunloopSDK } from '@runloop/api-client-ts';

    const runloop = new RunloopSDK();
    const devbox = await runloop.devbox.create({ name: 'my-devbox' });
    devbox.cmd.exec({ command: 'echo "Hello, World!"' });
    console.log(`Created devbox: ${devbox.id}`);
